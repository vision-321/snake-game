# 贪吃蛇游戏 Snake Game

一个使用纯 HTML5 + CSS3 + JavaScript 开发的经典贪吃蛇游戏，无需任何依赖，打开即玩。

## 游戏截图

```
🐍 贪吃蛇
按 空格 开始游戏
```

## 功能特性

### 游戏玩法
- 经典贪吃蛇规则，吃食物增长身体
- 撞墙或撞到自己则游戏结束
- 分数越高，蛇移动速度越快

### 难度选择
| 难度 | 初始速度 | 最高速度 | 每个食物得分 |
|------|----------|----------|--------------|
| 简单 | 慢 | 较慢 | 5 分 |
| 普通 | 中等 | 中等 | 10 分 |
| 困难 | 快 | 较快 | 15 分 |
| 地狱 | 极快 | 极快 | 20 分 |

### 主题皮肤
- **经典绿** - 传统贪吃蛇配色
- **霓虹蓝** - 赛博朋克风格
- **烈焰红** - 热情火焰主题
- **暗夜紫** - 神秘暗夜风格

### 音效系统
- 吃食物音效
- 游戏结束音效
- 8-bit 风格背景音乐
- 音效和音乐可独立开关

### 数据持久化
- 自动保存最高分
- 记住难度设置
- 记住皮肤偏好
- 记住音效开关状态

## 操作说明

| 按键 | 功能 |
|------|------|
| `↑` `↓` `←` `→` | 控制蛇移动方向 |
| `W` `A` `S` `D` | 控制蛇移动方向（备选） |
| `空格` | 开始游戏 / 暂停 / 继续 |
| `R` | 重新开始游戏 |
| `M` | 开关背景音乐 |

## 快速开始

1. 克隆仓库
```bash
git clone https://github.com/vision-321/snake-game.git
```

2. 打开游戏
```bash
cd snake-game
# 直接用浏览器打开 index.html
start index.html  # Windows
open index.html   # macOS
```

或者直接双击 `index.html` 文件即可开始游戏。

## 技术栈

- **HTML5 Canvas** - 游戏画面渲染
- **CSS3** - 界面样式和动画效果
- **JavaScript (ES6+)** - 游戏逻辑
- **Web Audio API** - 音效和背景音乐生成
- **LocalStorage** - 本地数据存储
